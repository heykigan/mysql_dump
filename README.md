# MySQL Dump 
Take SQL backup for projects and server where mysql dumps cannot be taken directly

## Installation
`$ npm install`

## Configuration

Please change the values of host,user,password,database,tables in app.js as

```
host: 'localhost',
user: 'username',
password: 'password',
database: 'database',
tables: ['tab1e1', 'table2', 'table3']

```


## Run the script  

`$ npm start`
